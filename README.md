# ParkSmart - BD II Proyecto #15
Sistema de gestión para parqueadero privado.

## Dominio
Parqueo / Tarifas / Abonos / Control

## Motores implementados
- MySQL
- PostgreSQL
- MS SQL Server
- Oracle

## Estructura del repositorio
- `docs/` → épicas, historias de usuario, guía completa
- `scripts/mysql/` → script SQL para MySQL
- `scripts/postgresql/` → script SQL para PostgreSQL
- `scripts/mssql/` → script SQL para MS SQL Server
- `scripts/oracle/` → script SQL para Oracle
- `images/` → capturas de diagramas ER en cada motor y gestor
